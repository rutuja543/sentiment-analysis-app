# 🧠 AI Sentiment Analysis Web App

## 📌 Overview

This project is a machine learning-based web application that analyzes user input text and predicts whether the sentiment is **positive or negative**.

The app is built using **TF-IDF and Logistic Regression** and deployed using **Gradio on Hugging Face Spaces**.

---

## 🚀 Features

* Sentiment prediction (Positive / Negative)
* Confidence score display
* Clean and interactive UI
* Real-time text analysis
* Deployed online for public access

---

## 🛠 Tech Stack

* Python
* Scikit-learn
* Gradio
* Hugging Face Spaces

---

## ⚙️ How It Works

1. Text is preprocessed (lowercase, punctuation removed)
2. TF-IDF converts text into numerical features
3. Logistic Regression predicts sentiment
4. Output shows sentiment + confidence score

---

## 📊 Model Performance

* Accuracy: ~85% (approx based on test split)
